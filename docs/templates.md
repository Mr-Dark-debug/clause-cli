# Project Templates

Clause CLI comes with built-in templates for popular stacks.

## Frontend Templates

### Next.js
- **Stack**: Next.js 14 (App Router), React, TypeScript, Tailwind CSS
- **Features**: 
  - Optimized project structure
  - SEO best practices configured
  - Jest/Vitest setup
  - Component library integration (shadcn/ui optional)

### React
- **Stack**: Vite, React, TypeScript
- **Features**:
  - Fast HMR
  - Minimal boilerplate
  - Router setup (React Router)

### Vue
- **Stack**: Vite, Vue 3, TypeScript
- **Features**:
  - Pinia state management
  - Vue Router

## Backend Templates

### FastAPI
- **Stack**: FastAPI, Pydantic, SQLAlchemy, Alembic
- **Features**:
  - Async by default
  - Auto-generated OpenAPI docs
  - Modular architecture (Service-Repository pattern)
  - JWT Authentication ready

### Express
- **Stack**: Node.js, Express, TypeScript
- **Features**:
  - Strict type checking
  - Middleware chains
  - Error handling middleware

### Django
- **Stack**: Django 5, DRF
- **Features**:
  - Batteries included
  - Admin interface
  - ORM
