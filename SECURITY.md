## Reporting a Vulnerability

We take the security of Clause CLI seriously. If you believe you have found a security vulnerability in Clause CLI, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to security@clause.dev.

You should receive a response within 48 hours. If for some reason you do not, please follow up via email to ensure we received your original message.

We prefer all communications to be in English.

## Disclosure Policy

1.  **Initial Report**: You report the vulnerability to us via email.
2.  **Assessment**: We assess the vulnerability and its impact.
3.  **Fix**: We develop a fix for the vulnerability.
4.  **Release**: We release a new version of Clause CLI with the fix.
5.  **Disclosure**: We publicly disclose the vulnerability and credit you for the discovery (if desired).
