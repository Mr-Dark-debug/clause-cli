Using a Template

bash
# List all available templates
clause init --list

# Use a specific template
clause init my-project --template react-app

# Use a template with defaults
clause init my-api --template go-rest-api -y

Creating Custom Templates

You can create custom templates and share them with your team. A template is a directory containing files and a template.yaml manifest.

yaml
# template.yaml
name: my-custom-template
description: My team's custom project template
version: 1.0.0
variables:
  - name: project_name
    prompt: Project name
    required: true
  - name: database
    prompt: Database type
    default: postgres
    options: [postgres, mysql, sqlite]
