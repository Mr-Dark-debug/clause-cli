CLI Reference

Complete command reference for the Clause CLI.

Usage

bash
clause [global flags] <command> [command flags] [arguments]

Global Flags

Flag Description
-c, --config <path> Config file path
-v, --verbose Verbose output
-q, --quiet Suppress non-essential output
--no-color Disable colored output
-h, --help Help for clause

clause init

Initialize a new Clause project with AI governance.

bash
clause init [project-name] [flags]

Flags

  • -t, --template <name> ‚Äî Use a specific template
  • --no-git ‚Äî Skip git initialization
  • --non-interactive ‚Äî Run without the TUI wizard
  • -y, --yes ‚Äî Accept all defaults

Examples

bash
# Interactive wizard
clause init my-project

# Use a specific template
clause init my-api --template go-rest-api

# Non-interactive with defaults
clause init my-app -y

clause add

Add components or governance rules to an existing project.

bash
clause add <type> [flags]

Types

  • governance ‚Äî Add a new governance rule file
  • component ‚Äî Register a new component
  • template ‚Äî Add files from a template

Examples

bash
# Add a governance rule
clause add governance --name testing-rules

# Register a component
clause add component --name UserService --path src/services/UserService.ts

clause validate

Validate that your project complies with its governance rules.

bash
clause validate [flags]

Flags

  • --strict ‚Äî Treat warnings as errors
  • --fix ‚Äî Auto-fix fixable issues

clause update

Update Clause CLI to the latest version.

bash
clause update

clause config

Manage Clause configuration.

bash
# View current config
clause config list

# Set a value
clause config set default_template go-rest-api

# Get a value
clause config get default_template

clause version

Print the installed version.

bash
clause version
# Output: clause version 1.0.0 (abc1234)

clause completion

Generate shell completion scripts.

bash
# Bash
clause completion bash > /etc/bash_completion.d/clause

# Zsh
clause completion zsh > "${fpath[1]}/_clause"

# Fish
clause completion fish > ~/.config/fish/completions/clause.fish

# PowerShell
clause completion powershell > clause.ps1
