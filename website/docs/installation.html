Installation

Install Clause on your platform in seconds. Choose the method that works best for you.

Requirements

  • Go 1.21+ ‚Äî required only if building from source
  • Git ‚Äî optional, for version control integration
  • A modern terminal / command prompt

Installation Methods

Homebrew (Recommended)

bash
brew install clause-cli/tap/clause

curl | bash

bash
curl -fsSL https://clause.dev/install.sh | bash

Manual Download

Download the latest release from GitHub Releases and move the binary to /usr/local/bin.

curl | bash

bash
curl -fsSL https://clause.dev/install.sh | bash

APT (Debian/Ubuntu)

bash
curl -fsSL https://clause.dev/apt/gpg.key | sudo gpg --dearmor -o /usr/share/keyrings/clause.gpg
echo "deb [signed-by=/usr/share/keyrings/clause.gpg] https://clause.dev/apt stable main" | sudo tee /etc/apt/sources.list.d/clause.list
sudo apt update && sudo apt install clause

Snap

bash
sudo snap install clause

Arch Linux (AUR)

bash
yay -S clause-cli

Winget

powershell
winget install Clause.ClauseCLI

Scoop

powershell
scoop bucket add clause-cli https://github.com/Mr-Dark-debug/scoop-bucket
scoop install clause

Chocolatey

powershell
choco install clause-cli

PowerShell Script

powershell
irm https://clause.dev/install.ps1 | iex

Build from Source

bash
git clone https://github.com/Mr-Dark-debug/clause-cli.git
cd clause-cli
make build
sudo mv bin/clause /usr/local/bin/

Verify Installation

bash
clause --version
# Output: clause version 1.0.0

Updating

bash
# Self-update
clause update

# Or via your package manager
brew upgrade clause

Uninstallation

bash
# Homebrew
brew uninstall clause

# Manual
rm /usr/local/bin/clause
